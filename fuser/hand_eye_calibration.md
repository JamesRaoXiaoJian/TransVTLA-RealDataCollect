## 二、为什么要做相机标定

### 2.1 内参标定的目的

内参标定确定相机的**内部几何模型**，用于：

| 用途 | 说明 |
|------|------|
| 畸变校正 | 广角镜头存在桶形/枕形畸变，需校正才能准确测量 |
| 3D 重建 | 将 2D 像素坐标反投影到 3D 空间 |
| 视觉测量 | 准确计算物体实际尺寸 |

### 2.2 外参标定的目的

外参标定确定**相机与机械臂/世界坐标系之间的空间变换关系**：

```
T_base_cam = T_base_ee · T_cam2ee
```

| 相机类型 | 标定方式 | 说明 |
|----------|----------|------|
| eye-on-hand（腕部） | 手眼标定（AX=XB） | 相机随末端执行器运动 |
| eye-to-hand（固定） | 直接测量 + 棋盘格验证 | 相机固定，求解相机在基座下的位姿 |

## 三、标定方法

### 3.1 内参标定（针孔模型 + 畸变系数）

使用 OpenCV `calibrateCamera` 求解：

```
K = [fx, 0, cx; 0, fy, cy; 0, 0, 1]     # 内参矩阵
D = [k1, k2, p1, p2, k3]                 # 畸变系数
```

**操作方法**：

```bash
python -c "
import cv2, numpy as np
# 拍摄 15~20 张不同角度的棋盘格图像
# 使用 findChessboardCorners 检测角点
# 调用 calibrateCamera 求解 K, D
"
```

### 3.2 手眼标定（eye-on-hand 相机）

**适用相机**：`franka_right`、`franka_front`

使用 OpenCV `calibrateHandEye` 求解 `AX = XB`：

- **A**：`T_base_ee`（末端执行器在基座坐标系下的位姿变化）
- **B**：`T_target`（棋盘格在相机坐标系下的位姿变化）
- **X**：`T_cam2ee`（相机在末端执行器坐标系下的位姿）

### 3.3 固定相机外参标定（eye-to-hand）

**适用相机**：`dji_action5`（DJI Action 5 Pro）

**方法：测量法**

1. 将 DJI Action 5 Pro 固定在基座左侧 20cm，高度 30cm
2. 用卷尺/卡尺测量相机光心在基座坐标系下的位置
3. 用水平仪/陀螺仪测量相机朝向（俯仰角、偏航角）
4. 将测量值转换为旋转矩阵 `R` 和平移向量 `t`
5. 用棋盘格验证：拍摄棋盘格并重投影，检查误差

**测量步骤**：

```
1. 确定基座坐标系原点（通常在机械臂底座中心）
2. 测量相机光心到原点的 x, y, z（米）
   → t = [x, y, z]
3. 测量相机朝向：
   - 相机朝向工作区域，使用手机指南针/水平仪测量偏航角 yaw
   - 粗略标定：假设相机水平，R 使用标准方向
   - 精确标定：用棋盘格拍摄多个位置，拟合 R
```

## 四、各相机标定步骤

### 4.1 RealSense D455（franka_right / franka_front）

> 标定板：A4 纸打印 11×8 棋盘格，方格尺寸 25mm
> 代码：`hand_eye_calibration.py`

**操作步骤**：

1. 打印棋盘格标定板，粘贴在平硬纸板上
2. 固定棋盘格，确保相机全程能看到
3. 运行采集程序：

```bash
python hand_eye_calibration.py --mode realsense
```

4. 推动机械臂末端（带动相机），按 **SPACE** 采集 15~20 组
5. 按 **S** 保存数据并计算结果
6. 将 `R_cam2ee` 和 `t_cam2ee` 填入 `camera.py`

### 4.2 DJI Action 5 Pro（dji_action5）

> 标定方法：内参预估 + 外参测量
> 相机位置：基座左侧 20cm，高度 30cm，朝向工作区域

#### 内参标定（预估，需实测验证）

| 参数 | 预估值 | 说明 |
|------|--------|------|
| 分辨率 | 1920×1080 | 常用视频模式 |
| fx | ~1000.0 | 等效焦距 |
| fy | ~1000.0 | 等效焦距 |
| cx | 960.0 | 图像中心 |
| cy | 540.0 | 图像中心 |
| 畸变系数 | 需实测 | 广角镜头畸变较明显 |

> ⚠️ **注意**：以上为粗估参数，实际内参需用棋盘格标定获取。
> 建议：DJI Action 5 Pro 支持 4K 录制，可查官方参数或实测棋盘格角点求解。

#### 外参标定（测量法）

**测量值**（待填入）：

```
相机在基座坐标系下的位置：
x = -0.20 m（基座左侧 20cm）
y =  0.30 m（高度 30cm）
z =  0.00 m（与基座同高）

朝向：相机朝向机械臂工作区域
（偏航角 yaw ≈ -90° 或根据实际朝向调整）
```

**测量步骤**：

```
1. 确定基座坐标系原点
2. 用卷尺测量：相机光心到原点的 x, y, z 距离
3. 用手机指南针测量相机光轴朝向，计算 yaw 角
4. 将 yaw 转换为旋转矩阵 R
5. 合并为外参 T_base_cam = [R|t]
6. 拍摄棋盘格验证重投影误差
```

## 五、标定结果填写

### 5.1 franka_right / franka_front（手眼标定结果）

```python
"franka_right": CameraParams(
    K=torch.tensor([
        [387.4, 0.0, 320.0],
        [0.0, 386.9, 240.0],
        [0.0, 0.0, 1.0]
    ], dtype=torch.float32),
    R=torch.tensor([
        [r11, r12, r13],   # 手眼标定结果
        [r21, r22, r23],
        [r31, r32, r33]
    ], dtype=torch.float32),
    t=torch.tensor([tx, ty, tz], dtype=torch.float32)
),
```

> `R` 和 `t` 是 **世界（基座）→相机** 的变换。
> 若标定得到 `T_cam2ee`，需代入代表性 `T_base_ee` 计算：
> `T_base_cam = T_base_ee @ T_cam2ee`

## 六、重投影误差评估

| 误差级别 | 旋转误差 | 平移误差 | 说明 |
|----------|----------|----------|------|
| 优秀 | < 0.5° | < 1mm | 标定非常精确 |
| 良好 | < 1° | < 5mm | 满足使用需求 |
| 一般 | 1°~2° | 5~10mm | 可接受，建议优化 |
| 较差 | > 2° | > 10mm | 需要重新标定 |
