## 7. 时间戳异常处理建议

对于存在时间戳异常的 session，建议在 RLDS 转换脚本中:

```python
# 在 _generate_examples 中添加时间戳检查
def check_timestamps(timestamps):
    """检查时间戳是否正常"""
    intervals = np.diff(timestamps)
    # 如果存在超过 10 秒的间隔，认为异常
    if np.any(intervals > 10e6):
        return False
    return True
```
